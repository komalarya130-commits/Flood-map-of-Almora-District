# Flood-map-of-Almora-District
# Almora Flood Mapping using Google Earth Engine

## Project Overview
This project detects flood affected areas in Almora district using Sentinel-1 SAR satellite data in Google Earth Engine.

## Techniques Used
- Sentinel-1 SAR
- Flood extraction
- Thresholding
- Noise removal
- GeoTIFF export

## Outputs
- Flood affected area map
- GeoTIFF export
- Vector flood analysis

## Tools
- Google Earth Engine
- QGIS
- Remote Sensing
- GIS Analysis
